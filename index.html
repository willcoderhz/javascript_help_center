Variables and Data Types

The fundamental building blocks in JavaScript, including data types such as strings, numbers, booleans, arrays, and objects, as well as how to declare and use variables.

Functions and Scope

Functions are the basic units in JavaScript for encapsulating functionality and logic. Learn how to create and call functions, function scope, and closures.

Learn how to use conditional statements (if-else, switch), loops (for, while), and exception handling (try-catch) to control the execution order of your code.

The Document Object Model (DOM) allows you to manipulate web page content using JavaScript. Learn how to access and modify elements, attributes, and styles, as well as handle events.

Asynchronous Programming

Understand how to work with asynchronous operations in JavaScript using callbacks, Promises, and async/await, such as AJAX requests, setTimeout, and other delayed operations.

Advanced Topics

Explore advanced concepts in JavaScript, such as prototype inheritance, modular programming, generators, and iterators, as well as learn about new ECMAScript features.
